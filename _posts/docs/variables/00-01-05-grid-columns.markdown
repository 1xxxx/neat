The `$grid-column` variable sets the total number of columns in the grid. Its value can be overidden inside a media query using the `media()` mixin.

### Syntax

{% highlight scss %}
$grid-columns: 8;
{% endhighlight %}

- *Value >* **number** *positive, unitless*
- *Default >* *12*
