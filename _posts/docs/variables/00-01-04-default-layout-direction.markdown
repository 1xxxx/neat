The `$default-layout-direction` variable sets the default layout direction of the grid.

### Syntax

{% highlight scss %}
$default-layout-direction: RTL;
{% endhighlight %}

- *Values >* `LTR | RTL` Left-to-right or right-to-left layouts
- *Default >* `LTR`
