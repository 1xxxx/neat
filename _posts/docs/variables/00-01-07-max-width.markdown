The `$max-width` variable sets the max-width property of the element that includes `outer-container()`.

### Syntax

{% highlight scss %}
$max-width: 1024px;
{% endhighlight %}

- *Value >* **number** *positive, unit*
- *Default >* `em(1088)` (See [Bourbon docs][http://bourbon.io/docs/#px-to-em])
