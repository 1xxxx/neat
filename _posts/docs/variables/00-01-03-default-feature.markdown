{% highlight sass %}
$default-feature
  type: string
  default: min-width
{% endhighlight %}

Sets the default [media feature](http://www.w3.org/TR/css3-mediaqueries/#media) that `media()` and `new-breakpoint()` revert to when only a breakpoint value is passed.
