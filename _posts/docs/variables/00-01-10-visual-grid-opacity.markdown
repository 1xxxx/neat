The `$visual-grid-opacity` variable sets the opacity property of the visual grid.

### Syntax

{% highlight scss %}
$visual-grid-opacity: 0.7;
{% endhighlight %}

- *Value >* **number** *positive, less or equal to 1*
- *Default >* `0.4`
